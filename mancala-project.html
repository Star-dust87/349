Mancala Game with AI

Completed: April 2025
Duration: 3 weeks
Role: Developer
Python AI Game Development Minimax Algorithm Alpha-Beta Pruning

Project Overview

This project is a command-line implementation of the classic board game Mancala with an intelligent AI opponent. The game features multiple modes including human vs. human, human vs. AI, and AI vs. AI gameplay. The AI uses the minimax algorithm with alpha-beta pruning to make strategic decisions at varying difficulty levels.

Mancala is one of the oldest known games in the world, dating back to ancient Egypt. The game is a perfect candidate for AI implementation because it involves strategic thinking and forward planning, making it an excellent demonstration of search algorithms and game theory.

Technical Implementation

The game is structured using object-oriented programming principles with two main classes:

  1. MancalaGame: Handles the game logic, board state, and player interactions.
  2. MancalaAI: Implements the AI opponent using the minimax algorithm.

The minimax algorithm with alpha-beta pruning allows the AI to look ahead several moves and choose the optimal play. The evaluation function considers multiple factors including:

  • Seeds in the player's store (main scoring mechanism)
  • Seeds in the player's pits (available moves)
  • Empty pits (potential for capture)
  • Strategic pit positions (pits closer to the store are weighted higher)

AI Implementation Highlight

The core of the AI decision-making process is implemented in the minimax function:

def minimax(self, board, depth, alpha, beta, maximizing_player, player):
    """Minimax algorithm with alpha-beta pruning"""
    if depth == 0 or checkForWinner(board) != 'no winner':
        return self.evaluate_board(board, player), None

    valid_moves = self.get_valid_moves(board, '1' if maximizing_player else '2')
    if not valid_moves:
        return self.evaluate_board(board, player), None

    best_move = valid_moves[0]
    if maximizing_player:
        max_eval = float('-inf')
        for move in valid_moves:
            board_copy = deepcopy(board)
            next_player = makeMove(board_copy, '1', move)
            eval_score, _ = self.minimax(board_copy, depth - 1, alpha, beta, 
                                      next_player == '1', player)
            if eval_score > max_eval:
                max_eval = eval_score
                best_move = move
            alpha = max(alpha, eval_score)
            if beta <= alpha:
                break
        return max_eval, best_move
    else:
        min_eval = float('inf')
        for move in valid_moves:
            board_copy = deepcopy(board)
            next_player = makeMove(board_copy, '2', move)
            eval_score, _ = self.minimax(board_copy, depth - 1, alpha, beta,
                                      next_player == '1', player)
            if eval_score < min_eval:
                min_eval = eval_score
                best_move = move
            beta = min(beta, eval_score)
            if beta <= alpha:
                break
        return min_eval, best_move

Challenges & Solutions

One of the main challenges in implementing this game was creating an effective evaluation function for the AI. The game of Mancala has many subtle strategic elements that are difficult to quantify. I addressed this by:

  1. Analyzing existing Mancala strategies and common patterns.
  2. Implementing a weighted scoring system that considers multiple factors.
  3. Testing and refining the evaluation function through repeated gameplay.
  4. Adding positional weighting to prioritize certain pits over others.

Another challenge was optimizing the AI to make decisions in a reasonable amount of time. The minimax algorithm's complexity grows exponentially with depth, so I implemented alpha-beta pruning to significantly reduce the number of nodes evaluated.

Game Flow

The gameplay experience follows this sequence:

  1. Game starts with a welcome message and rules display.
  2. Player selects game mode (human vs. human, human vs. AI, AI vs. AI).
  3. If playing against AI, player selects difficulty level.
  4. The game loop begins, displaying the board and prompting for moves.
  5. Players take turns until one side has no seeds left.
  6. Final score is calculated and a winner is declared.
